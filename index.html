Step 01

Create a Spotify Developer App

Spotify needs a free Developer App to give you secure access. It takes about 3 minutes.

  1. Open developer.spotify.com/dashboard
  2. Log in with your Spotify account, click Create App
  3. Set App name and App description (e.g. "CleanPlay")
  4. For Redirect URI, paste exactly:
  1. Tick Web API under "Which API/SDKs are you planning to use?"
  2. Accept terms → Save
  3. Open the app → Settings → copy the Client ID
Important: Keep the app in Development Mode. Only Spotify accounts you explicitly add (Settings → User Management) can log in. This is your first security layer.
Step 02

Enter your Client ID

Paste the Client ID from your Spotify app dashboard.

Step 03 — recommended

Lock to your Spotify account

The strongest security layer. After login, the app verifies the Spotify user matches yours exactly. Even if someone got past the password and your dev app, they couldn't use it with a different account.

Find your username at spotify.com/account — look for "Username".
